# Game-Tracker
Программа для отслеживания активности в играх

## О проекте
WinForms приложение для удобного отслеживания активности в играх. Представляет собой менджер, который собирают статистику об играх, _**запущенных из самого приложения!**_

## Технологии
- C#
- Windows Forms
- .NET

## 📁 Структура проекта

**WinFormsApp_GameTracker/**

- `Program.cs` - Точка входа в приложение
- `MainForm.cs` - Главная форма (список игр)
- `MainForm.Designer.cs` - Дизайн главной формы
- `MainForm.resx` - Ресурсы главной формы
- `GameInfoForm.cs` - Форма информации об игре
- `GameInfoForm.Designer.cs` - Дизайн формы информации
- `GameInfoForm.resx` - Ресурсы формы информации
- `DataManager.cs` - Управление данными (загрузка/сохранение)
- `GameInfo.cs` - Модель данных игры (название, путь, исполняемый процесс)
- `GameRepository.cs` - Репозиторий для работы с играми (CRUD)
- `GameStats.cs` - Статистика активности (время)

## Особенности реализации
- Автоматическое сканирование папки C:/GOG Games;
- Фоновый сбор статистики при запуске игр;
- Сохранение данных в JSON формате;
Для приожения разработан интерфейс с помощью библиотеки Windows Forms платформы .NET. Интерефейс позволяет:
- Видеть игры, расположенные в папке компьютера GOG Games корня каталога C:/ (по умолчанию);
- Запускать игры;
- Просматривать активность в играх.

## Установка
1. Клонировать репозиторий;
2. Открыть .sln файл в Visual Studio;
3. Собрать проект (Ctrl+Shift+B);

## 🚀 Возможности развития
1. При большом количестве игр можно организовать поиск по названию;
2. Добавить графики статистики;
3. Добавить возможность сохранять статистику в файлы.
